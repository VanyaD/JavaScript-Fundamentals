Problem 4. Maximal increasing sequence

Write a script that finds the maximal increasing sequence in an array.

Examples:

  • [3, 1, 1, 1, 2, 3, 3, 2, 2, 2, 1]
  • [7, 5, 6, 7, 8, 9, 10, 12, 3, 5, 5]
  • [9, 10, 11, 10, 10, 7, 7, 7, 7, 7]
  • [10, 1, 2, 3, 4, 5]
You could use Chrome console (F12) or Firebug to see the results.
